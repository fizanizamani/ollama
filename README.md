# Ollama AI Projects Collection

## Overview

This repository contains a collection of AI applications built using Ollama, Python, FastAPI, Streamlit, React, FAISS, and Vector Databases. These projects explore conversational AI, Retrieval-Augmented Generation (RAG), multi-agent systems, note-taking assistants, and multilingual chatbot experiences.

The purpose of this repository is to showcase practical AI development skills, local LLM deployment, backend API development, vector search implementation, and modern AI application architecture.
